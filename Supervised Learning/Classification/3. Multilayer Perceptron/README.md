## About

This Jupyter notebook applies the Multilayer perceptron neural network to model both the MNIST number dataset and the MNIST fashion dataset. We then test and compute the testing errors for both datasets. Next, we plot the error for different learning rates and hidden network sizes, and display the misidentified testing data. Finally, we conclude that the mutlilayer perceptron neural network models both datasets well.

## Data

We are using the MNIST dataset, which consists of 70,000 images given by 28x28 matrices of numbers representing grayscale images.

The first dataset consists of images of numbers, and the second dataset consists of images of various articles of clothing.

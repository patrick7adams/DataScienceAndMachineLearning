# Data Science and Machine Learning

By Patrick Adams

## About

This section of the repository implements a variety of supervised learning algorithms for both classification and regression algorithms. It then plots results and discusses the errors of individual algorithms.

## Data

The section of the repository uses four datasets, each explained in local README files: the AutoMPG dataset, the Titanic dataset, the MNIST number dataset, and the MNIST fashion dataset.

## Prerequisites

This package was built for Python 3.12.

The following packages are necessary to be installed on your system:
- numpy
- pandas
- seaborn
- matplotlib
- jupyter
- mlxtend
- scikit-learn
